install_YouCompleteMe_for_VIM1.python:sudo apt-get install python python3 python-dev python3-dev2.ncurses:sudo apt-get install libncurses5 libncurses5-dev3.Build vim:./configure --with-features=huge --enable-pythoninterp --enable-python3interp --enable-luainterp --enable-multibyte --enable-sniff --enable-fontsetNotes:check if you really need to do so. In vim, :echo has('python') return 1 means your current vim supports python, otherwise you need to build vim from scratch.  4.vundle:git clone https://github.com/gmarik/vundle.git ~/.vim/bundle/vundle5.llvm-clang:please refer to  install_llvm-clang_on_linux6. build YouCompleteMecd ~/.vim/bundle/YouCompleteMe./install.sh --clang-completer --omnisharp-completerNotes:--omnisharp is used for c#, not used for linux.
